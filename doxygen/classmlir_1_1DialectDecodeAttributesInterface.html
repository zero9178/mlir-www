Registered hook to decode opaque constants associated with this dialect.

The hook function attempts to decode an opaque constant tensor into a tensor with non-opaque content. If decoding is successful, this method returns success() and sets 'output' attribute. If not, it returns failure() and leaves 'output' unspecified. The default hook fails to decode.
