Detailed Description

template<typename ValueT>
class mlir::LatticeElement< ValueT >

This class represents a lattice holding a specific value of type ValueT.

Lattice values (ValueT) are required to adhere to the following:

  • static ValueT join(const ValueT &lhs, const ValueT &rhs);
    • This method conservatively joins the information held by lhs and rhs into a new value. This method is required to be monotonic.
  • static ValueT getPessimisticValueState(MLIRContext *context);
    • This method computes a pessimistic/conservative value state assuming no information about the state of the IR.
  • static ValueT getPessimisticValueState(Value value);
    • This method computes a pessimistic/conservative value state for value assuming only information present in the current IR.
  • bool operator==(const ValueT &rhs) const;
