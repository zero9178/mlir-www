333  /// Updates the constraints to be the smallest bounding (enclosing) box that
334  /// contains the points of `this` set and that of `other`, with the symbols
335  /// being treated specially. For each of the dimensions, the min of the lower

346  /// output = {s0 + 1 <= d0 <= s0 + 20}
