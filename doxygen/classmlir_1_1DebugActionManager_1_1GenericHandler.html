Detailed Description

This class represents a generic action handler.

A generic handler allows for handling any action type. Handlers of this type are useful for implementing general functionality that doesn't necessarily need to interpret the exact action parameters, or can rely on an external interpreter (such as the user). Given that these handlers are generic, they take a set of opaque parameters that try to map the context of the action type in a generic way.
