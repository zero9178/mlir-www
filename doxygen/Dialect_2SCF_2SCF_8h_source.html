62 /// Creates a perfect nest of "for" loops, i.e. all loops but the innermost
63 /// contain only another loop and a terminator. The lower, upper bounds and
64 /// steps are provided as `lbs`, `ubs` and `steps`, which are expected to be of
65 /// the same size. `iterArgs` points to the initial values of the loop iteration
66 /// arguments, which will be forwarded through the nest to the innermost loop.
67 /// The body of the loop is populated using `bodyBuilder`, which accepts an
68 /// ordered list of induction variables of all loops, followed by a list of
69 /// iteration arguments of the innermost loop, in the same order as provided to
70 /// `iterArgs`. This function is expected to return as many values as
71 /// `iterArgs`, of the same type and in the same order, that will be treated as
72 /// yielded from the loop body and forwarded back through the loop nest. If the
73 /// function is not provided, the loop nest is not expected to have iteration
74 /// arguments, the body of the innermost loop will be left empty, containing
75 /// only the zero-operand terminator. Returns the LoopNest containing the list
76 /// of perfectly nest scf::ForOp build during the call.
77 /// If bound arrays are empty, the body builder will be called
78 /// once to construct the IR outside of the loop with an empty list of induction
79 /// variables.
80 LoopNest buildLoopNest(
81  OpBuilder &builder, Location loc, ValueRange lbs, ValueRange ubs,
82  ValueRange steps, ValueRange iterArgs,
83  function_ref<ValueVector(OpBuilder &, Location, ValueRange, ValueRange)>
84  bodyBuilder = nullptr);
