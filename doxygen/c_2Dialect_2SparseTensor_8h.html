enum MlirSparseTensorDimLevelType

Dimension level types that define sparse tensors:

  • MLIR_SPARSE_TENSOR_DIM_LEVEL_DENSE - dimension is dense, every entry is stored
  • MLIR_SPARSE_TENSOR_DIM_LEVEL_COMPRESSED - dimension is sparse, only nonzeros are stored.
  • MLIR_SPARSE_TENSOR_DIM_LEVEL_SINGLETON - dimension contains single coordinate, no siblings.

These correspond to SparseTensorEncodingAttr::DimLevelType in the C++ API. If updating, keep them in sync and update the static_assert in the impl file.
