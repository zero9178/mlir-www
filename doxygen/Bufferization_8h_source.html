35 /// Try to cast the given ranked MemRef-typed value to the given ranked MemRef
36 /// type. Insert a reallocation + copy if it cannot be statically guaranteed
37 /// that a direct cast would be valid.
38 ///
39 /// E.g., when casting from a ranked MemRef type with dynamic layout to a ranked
40 /// MemRef type with static layout, it is not statically known whether the cast
41 /// will succeed or not. Such `memref.cast` ops may fail at runtime. This
42 /// function never generates such casts and conservatively inserts a copy.
43 ///
44 /// This function returns `failure()` in case of unsupported casts. E.g., casts
45 /// with differing element types or memory spaces.
46 FailureOr<Value> castOrReallocMemRefValue(OpBuilder &b, Value value,
47  MemRefType type);
