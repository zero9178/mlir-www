SmallVector<SMLoc, 1> mlir::detail::SymbolState::nestedParserLocs

A set of locations into the main parser memory buffer for each of the active nested parsers.

Given that some nested parsers, i.e. custom dialect parsers, operate on a temporary memory buffer, this provides an anchor point for emitting diagnostics.
